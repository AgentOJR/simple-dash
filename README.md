# Personal Dashboard for macOS

A native macOS dashboard application that displays:
- GitHub contribution graph
- Recent GitHub repositories
- Quick launcher for common development applications

## Features
- Display your GitHub contribution activity
- Quick access to your recent repositories
- Launch common programming tools like Ghostty and Cursor
- Lives in your menu bar for easy access

## Requirements
- macOS 12.0 or later
- Xcode 13.0 or later for development
- GitHub personal access token for API access

## Setup
1. Clone this repository
2. Open the project in Xcode
3. Add your GitHub token in the settings
4. Build and run the application
